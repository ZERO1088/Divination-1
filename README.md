- # 占卜 Divination

  沉浸式数字占卜体验，进入精神空间，探索命运之谜。

  ## 空间入口

  ### 虚空之门 HomePortal

  主入口，两个精神空间的选择界面。采用去导航化设计，取消顶部横栏，让用户"进入"精神空间而非切换功能。

  - **Tarot Space** - 深紫罗兰色调，月光弥散，缓慢飘动的淡紫色光点
  - **I Ching Space** - 深墨色青灰渐变，晨曦微光，缓慢旋转的线条

  ### 易经六爻空间

  新中式暗调国风设计，通过抛掷三枚铜钱生成卦象。

  - **乾隆通宝铜钱** - 正面"乾"字，背面太极阴阳
  - **左右分栏布局** - 左侧铜钱+卦象，右侧解读区域
  - **擦灰动效** - 六爻完成后，解读内容渐次显现
  - **标准化爻线** - 阳爻实线、阴爻断线、动爻朱砂红高亮

  ### 塔罗牌空间

  暗黑神秘氛围，二十二张大阿卡纳专属 SVG 艺术设计。

  - **黑金暗调** - 深紫黑底色 + 金色线条
  - **命运舞台布局** - 大阿卡纳半圆弧排列，聚光灯效果
  - **三步流程** - 问题输入 → 抽卡选牌 → 单牌解读

  ## 目录结构

  ```
  src/
  ├── components/
  │   ├── IChing/        # 易经六爻页面
  │   ├── Landing/       # 虚空之门入口
  │   └── Tarot/         # 塔罗牌组件
  ├── constants/         # 卦象数据、塔罗牌数据
  ├── context/           # 全局状态管理
  ├── hooks/             # 自定义 hooks
  ├── pages/             # 页面组件
  ├── types/             # TypeScript 类型定义
  └── utils/             # 工具函数
  ```

  ## 路由

  | 路径                       | 页面                   |
  | -------------------------- | ---------------------- |
  | `/`                        | HomePortal（虚空之门） |
  | `/iching-question`         | 六爻问题输入           |
  | `/iching`                  | 六爻占卜               |
  | `/tarot-question`          | 塔罗问题输入           |
  | `/tarot-select`            | 塔罗抽卡选牌           |
  | `/tarot-interpret/:cardId` | 塔罗单牌解读           |

  ## 运行

  ```bash
  # 安装依赖
  pnpm install
  
  # 开发预览
  pnpm exec vite --host 0.0.0.0 --port 5000
  
  # 生产构建
  pnpm run build
  ```

  ## 设计理念

  - **去导航化** - 取消顶部横栏，进入"精神空间"而非切换功能
  - **沉浸感** - 每个入口是独立空间，有独特的光影、粒子和动效
  - **仪式感** - 从首页到占卜空间的过渡是"进入"而非"切换"
